# modac-bash

## Installation

source this folder into your bashrc
```bash
[ -f ~/.modac-bash-open/bashrc.sh ] && source ~/.modac-bash-open/bashrc.sh
```
for zsh use bash completion compatibility mode
```bash
autoload bashcompinit
bashcompinit
[ -f ~/.modac-bash-open/bashrc.sh ] && source ~/.modac-bash-open/bashrc.sh
```
